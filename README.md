## Repository Description
This is the repository for University of Washington, Husky Math Club. 
The website contains description of Husky Math Club @ University of Washington, showcases of activities, club events, officer information, and 
the primary methods of communications between management and members. 

## Maintenance & Updates
This repository is to be exclusively updated by staff and officers of Husky Math Club @ Univerisity of Washington, it will be maintenanced and 
monitored as the club evolves and changes, for the latest updates and information simply visit the active URL linked at the very top of this document. 

## Technology Stack 
This website is built with React and styled with Material UI, it is hosted on Netlify services. Please be sure to follow the norms, formats, and 
styling choices when making changes and updates to this website, please test your changes in another branch before merging with the production 
main branch, thank you in advance!
